Cabecera

Charset

Es una especificación que inidca cómo se deben interpretar y mostrar los caracteres en una página web.

Autor

Es una etiqueta que se usa para especificar al autor de la página web.

Description

Se utiliza para proporcionar una breve dscripción del contenido de una página web.

Keywords

Es para especificar una lista de palabras clave que describen el contenido de una página web.
